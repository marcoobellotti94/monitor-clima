Sobre o sistema
Como funciona
O robô roda toda segunda-feira às 6h automaticamente via GitHub Actions, sem precisar de nenhuma ação manual.
Para cada localidade, coleta dados de previsão (próximos 16 dias) e histórico (últimos 35 dias) de múltiplas fontes.
Os dados são armazenados no repositório GitHub e o dashboard é atualizado automaticamente após a coleta.
Bases de dados — Previsão
Open-Meteo — Fonte principal. API gratuita e open-source que integra modelos globais (GFS, ICON, ERA5). Resolução de 5–11 km. Previsão de até 16 dias.
Climatempo — Fonte secundária de validação cruzada. 30 anos de operação no Brasil, utiliza modelo europeu ECMWF. Previsão de até 15 dias.
Bases de dados — Histórico
INMET — Dados observados de estações físicas nos municípios. Fonte prioritária onde disponível. Estações: A213 (Tomé-Açu), A215 (Marajó leste), A325 (S.R. Nonato), A341 (Picos), A253 (Altamira).
ERA5 / ECMWF — Reanálise climática global do Centro Europeu de Previsão. Resolução de 9 km, disponível desde 1950. Usado como fallback onde não há estação INMET.
Semáforo de campo
Favorável — Chuva prevista abaixo de 10 mm. Condições adequadas para operações com máquinas pesadas.
Atenção — Acumulado entre 10 mm e 20 mm. Chuva contínua pode comprometer a sustentação do solo para tratores. Avaliar antes de operar.
Pausa recomendada — Acumulado acima de 20 mm. Risco elevado de atolamento, derrapagem e compactação excessiva do solo. Paralisar máquinas pesadas.
