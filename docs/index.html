Sobre o sistema
Como funciona
O robô coleta dados automaticamente toda segunda-feira às 6h via GitHub Actions, sem necessidade de ação manual.
Para cada localidade, são coletados previsão (16 dias) e histórico (35 dias) de múltiplas fontes independentes.
Os dados ficam no GitHub e o dashboard é atualizado automaticamente após cada coleta.
Fontes de previsão
ECMWF IFS (9 km) — Modelo europeu, o mais preciso globalmente para precipitação tropical. Aberto em out/2025.
GFS / ICON — Modelos americano e alemão usados para avaliar convergência entre previsões.
Climatempo — Fonte brasileira com validação humana de meteorologistas. Complemento regional.
Fontes de histórico
INMET — Estações físicas nos municípios. Dado observado real, prioritário onde disponível. Estações: A213, A215, A325, A341, A253.
ERA5 / ECMWF — Reanálise climática global (9 km, desde 1950). Usado onde não há estação física próxima.
Semáforo de campo
Favorável — Chuva prevista abaixo de 10 mm. Condições adequadas para máquinas pesadas.
Atenção — Entre 10 mm e 20 mm. Solo pode perder sustentação para tratores. Avaliar antes de operar.
Pausa recomendada — Acima de 20 mm. Risco elevado de atolamento e compactação. Paralisar máquinas pesadas.
